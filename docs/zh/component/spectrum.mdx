# Spectrum 频谱

Spectrum 组件是一个用于音频数据可视化的高级、可定制的工具，它底层基于 D3.js，来渲染交互式和动态的频谱显示。这个组件具有高度的可适应性，非常适合需要详细和精确音频频率分析的应用

## 特性

Spectrum 是使用快速傅里叶变换（FFT）的大小进行频谱分析的，它可以在频谱图上显示音频的频率和能量
