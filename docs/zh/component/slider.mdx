# Slider 滑动输入条

可以使用滑动输入条来控制数值范围，`Slider` 通常用于调节音量、亮度等

<Uncontrolled />

在非受控模式下，`Slider` 组件不可进行交互，可以使用该模式实现音量、效果器等数值的动态变化 (`hideThumb` `prohibitInteraction`)

## 标签 Data

- `data-dragging`: 是否处于拖拽状态

- `data-bilateral`: 是否为双向模式

- `data-vertical`: 是否为垂直模式

- `data-disabled`: 是否禁用

- `data-direction`: 处于中心的正向还是负向，`positive` 为正时，`negative` 为负时（仅在 `bilateral` 模式下可变）

## API

:::warning

`Slider` 组件继承了 `React.HTMLAttributes<HTMLDivElement>` 而不是原生的 `input`

:::
