# Knob 旋钮

可以使用旋钮组件对音量、效果器等参数进行调节

## 引入

```ts
import { Knob } from 'echo-ui'
```

### 尺寸相关

<Size />

为了不影响组件的正常使用，`Knob` 并没有将 `style` 和 `className` 的样式修改权限全部暴露出去，而是提供了一套尺寸相关的 props，可以通过这些 props 来调整组件的尺寸

`size` `trackWidth` `pointerWidth` `pointerHeight`

### 颜色相关

<Color />

与尺寸的修改相同， `Knob` 的颜色相关的样式也是通过 props 暴露出去的，可以通过这些 props 来调整组件的颜色

`trackColor` `progressColor` `buttonColor` `pointerColor`
