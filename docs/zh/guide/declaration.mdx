# 声明

## 关于组件

**Echo UI** 的组件库并不是一个完整的 Web 应用组件库，它并不包含表单、列表等组件。并且例如按钮、输入框等组件所暴露的 API 也是为了音频应用的交互而设计的。当然，你也可以与其他更为丰富的 UI 组件库搭配使用（推荐 [NextUI](https://nextui.org/docs/guide/introduction)、[shadcn/ui](https://ui.shadcn.com/)）

## 关于 Hook

在 Echo UI 中的绝大部分 Hook 都是基于 Tone.js 进行的封装，因此在使用 Hook 之前先阅读 [Tone.js 的文档](https://tonejs.github.io/) 会有更好的开发效率。当然，在构建使用组件的过程中，也完全可以使用任意的 API 库或者原生的 Web Audio API
