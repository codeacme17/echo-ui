## 传参

### `UseSpectrogramProps`

- **fftSize** : `number`

  FFT 的大小，必须是 2 的幂次方，默认为 `1024` (范围： 16 ~ 16384)

- **onReady**: `() => void`

  当 `analyser` 初始化完成后的回调函数

- **onError**: `() => void`

  当 hook 内部出现错误时的回调函数

## 返回值

### `Object`

- **init** : `() => void`

  调用该方法可以初始化 `analyser`，在初始化完成后会调用 `onReady` 回调函数

- **analyser** : `Tone.Analyser`

  初始化后构建的 [Tone.Analyser](https://tonejs.github.io/docs/14.7.77/Analyser) 实例

- **data** : `SpectrogramDataPoint[]`

  频谱图数据，可以自行定时通过 `getData` 获取或者使用 `observe` 方法自动开启监听

- **fftSize** : `number`

  FFT 的大小

- **getData** : `() => void`

  获取频谱图数据，在调用后会更新 `data` 的值，可以使用定时器或 `requestAnimationFrame` 进行调用

- **setFftSize** : `(fftSize: number) => void`

  设置 FFT 的大小

- **observe** : `(callback: () => void) => void`

  监听频谱图数据的变化，使用该函数可开启监听，并实时的获得 data 的变化

- **cancelObserve** : `() => void`

  取消监听

- **error**: `boolean`

  是否出现出错

- **errorMessage**: `string`

  出错的错误信息
